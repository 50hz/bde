Detailed Description

template<class TYPE>
class bdlb::NullInputIterator< TYPE >

Provide an input iterator that iterates over an empty sequence. All null iterators compare equal. Since the iteration sequence is empty, incrementing or de-referencing this iterator yields undefined behavior.
