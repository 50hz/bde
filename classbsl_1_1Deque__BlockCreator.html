Detailed Description

template<class VALUE_TYPE, class ALLOCATOR>
class bsl::Deque_BlockCreator< VALUE_TYPE, ALLOCATOR >

Allocates blocks at the beginning or end of a deque and tentatively adds them to the deque. Keeps track of how many of the newly allocated blocks have actually been used by the deque. Destructor automatically frees any unused blocks (e.g., in case an exception is thrown).
