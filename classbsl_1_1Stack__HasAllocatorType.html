Detailed Description

template<class CONTAINER>
class bsl::Stack_HasAllocatorType< CONTAINER >

This class computes a public constant VALUE, which is true if the passed CONTAINER defines a type CONTAINER::allocator_type and false otherwise. This is used in conjunction with enable_if to make methods of stack that take allocator arguments exist if CONTAINER::allocator_type is present, and remove them from the constructor overload set otherwise.
