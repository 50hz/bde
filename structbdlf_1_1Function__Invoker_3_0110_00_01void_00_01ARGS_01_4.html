Detailed Description

template<class ARGS>
struct bdlf::Function_Invoker< 10, void, ARGS >

This implementation-private class defines an invoker specialized for functions that return void and accept ten arguments of type A1, A2, A3, A4, A5, A6, A7, A8, A9, A10 respectively.
