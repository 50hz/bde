Detailed Description

template<typename TYPE>
struct bsl::add_volatile< TYPE >

This struct template implements the add_volatile meta-function defined in the C++11 standard [meta.trans.cv] to provide an alias type that has the same type as the (template parameter) TYPE except that the top-level volatile-qualifier has been added, unless TYPE is a reference, a function, or already volatile-qualified at the top-level.
