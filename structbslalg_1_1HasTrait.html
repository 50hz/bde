Detailed Description

template<typename TYPE, typename TRAIT>
struct bslalg::HasTrait< TYPE, TRAIT >

This meta-function evaluates to bslmf::MetaInt<1> if the parameterized type TYPE has the parameterized TRAIT, and to bslmf::MetaInt<0> otherwise. Note that TYPE "has" the TRAIT if bslalg_TypeTraits<TYPE> directly includes TRAIT or else includes a trait that implies TRAIT.
