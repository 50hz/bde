Detailed Description

template<class TYPE>
struct bslmf::ConstForwardingType< TYPE >

Use ConstForwardingType<TYPE> for forwarding an rvalue of the given TYPE. Note that for lvalues and pointer types, ConstForwardingType<TYPE> should produce the same type as the ForwardingType<TYPE>.
