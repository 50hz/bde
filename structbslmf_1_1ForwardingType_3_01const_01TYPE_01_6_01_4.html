Detailed Description

template<class TYPE>
struct bslmf::ForwardingType< const TYPE & >

This specialization is used when the template parameter of ForwardingType is a const T& and T is not a volatile type. In that case, it may be advantageous to pass an argument by value if T is a basic type, but not if it is a user-defined type.
