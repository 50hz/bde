Detailed Description

template<typename TYPE>
struct bslmf::IsClass< TYPE >

This metafunction derives from bsl::true_type if the specified TYPE is a class type, or is a reference to a class type, and from bsl::false_type otherwise.

Note that although this struct is functionally identical to bsl::is_class, and the use of bsl::is_class should be preferred.
