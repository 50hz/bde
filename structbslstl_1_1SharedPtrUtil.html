template<class TARGET , class SOURCE >
static bsl::shared_ptr<TARGET> bslstl::SharedPtrUtil::dynamicCast ( const bsl::shared_ptr< SOURCE > &  source  )  [static]

Return a bsl::shared_ptr<TARGET> object sharing ownership of the same object as the specified source shared pointer to the parameterized SOURCE type, and referring to dynamic_cast<TARGET *>(source.ptr()). If source cannot be dynamically cast to TARGET *, then an empty bsl::shared_ptr<TARGET> object is returned.
